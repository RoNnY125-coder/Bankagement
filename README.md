# Bankagement 💳

Bankagement is a mobile bank management app prototype designed to simulate a modern fintech experience.

This project focuses on UI/UX clarity, financial data presentation, and structured transaction flows.

---

## 🚀 Overview

Bankagement was built as a high-fidelity mobile banking prototype using:

- Stitch AI (for structured UI generation)
- Gemini 3 (for flow refinement and UX logic improvements)

The final interface was manually customized and refined to improve hierarchy, usability, and trust-driven design.

---

## 🎯 Key Features

### 🔐 Authentication Flow
- Login / Registration UI
- OTP verification screen
- Biometric login concept
- Form validation structure

### 🏠 Dashboard
- Account holder details
- Masked account number
- Available balance display
- Quick actions (Send Money, Pay Bills, Scan & Pay)
- Recent transactions preview

### 💸 Money Transfer Flow
- Recipient bank details input
- IFSC & account validation logic structure
- Transaction summary confirmation
- Success / Failure state screens

### 💳 Cards Section
- Virtual debit/credit card UI
- Freeze/Unfreeze toggle
- Spending limit concept

### 📊 Additional Sections
- Transaction history with filters
- Loan overview & EMI calculator UI
- Profile & security settings
- Dark / Light mode variations

## 🛠 Tech & Tools

This is a prototype project focused on UI/UX.

Tools used:
- Stitch AI (UI generation)
- Gemini 3 (flow optimization & refinement)
- Manual UI customization & layout restructuring

No real banking backend is connected. All data is mock data for demonstration purposes.
